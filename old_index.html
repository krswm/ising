Interval

The program calculates the expected value of a physical quantity \(f(t)\) (\(E^2\), \(E\), \(M^2\), or \(M\)) as: \[ \langle f \rangle = \frac{1}{A} \sum_{a = 0}^{A - 1} f(t - a \, \Delta t) \]

Autorun

The program automatically runs the simulation while changing the temperature to obtain the statistics.
